# 插件

ChatNeo 提供事件驱动的插件系统。在 **设置 > 插件** 中管理。

## 自动生成标题

AI 收到回复后自动为对话生成简短标题。

配置项：

| 字段 | 说明 |
|------|------|
| 触发时机 | 关闭 / 首条消息后 / 每条消息后 |
| 提供商 | 用于生成标题的提供商（可选，默认使用当前对话模型） |
| 模型 | 用于生成标题的模型 |

标题限制为 20 字以内。

## 上下文压缩

对话消息数超过阈值时，自动建议压缩历史消息。

配置项：

| 字段 | 默认值 | 说明 |
|------|--------|------|
| 阈值 | - | 触发压缩建议的消息数（6 - 100） |
| 提供商 | - | 用于总结的提供商（可选） |
| 模型 | - | 用于总结的模型 |

压缩过程：
1. 保留最近 4 条消息
2. 将早期消息发送给 AI 生成摘要
3. 摘要保存到对话中，后续作为上下文使用
4. 显著减少 Token 消耗
