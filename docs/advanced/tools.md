# 内置工具

ChatNeo 内置多种工具，让 AI 可以执行实际操作。在 **设置 > 工具** 中管理。

## 工具列表

### 网页搜索

搜索互联网获取最新信息。支持多种搜索引擎：

| 引擎 | 需要 API Key | 说明 |
|------|:---:|------|
| Brave | 否 | 免费，HTML 抓取 |
| Brave Search API | 是 | 官方 API |
| DuckDuckGo | 否 | 免费，HTML 抓取 |
| Google Custom Search | 是 | 需要 API Key + CX ID |
| Bing | 是 | 微软搜索 API |
| Tavily | 是 | AI 搜索 API |
| Jina AI | 是 | AI 搜索 |
| Exa | 是 | 语义搜索 |
| SearXNG | 否 | 自建搜索引擎 |

搜索结果会以引用格式 `[1][2]` 附在 AI 回答中。

### 网页阅读

抓取并解析指定 URL 的网页内容。

配置项：
- 最大字符数：4000 - 32000（默认 8000）
- 超时时间：5 - 60 秒

### 文件解析

读取并解析本地文件，支持：
- PDF
- Excel（xlsx / xls / xlsm / ods）
- Word（docx）
- 纯文本（txt / csv / json / md）

配置项：最大文件大小 1 - 100 MB（默认 10 MB）。

### 计算器

计算数学表达式，支持加减乘除、取模、幂运算和括号。

### 代码运行

在本地沙盒环境中运行代码。

支持语言：Python、JavaScript、Shell。

配置项：
- 可用语言选择
- 超时时间：5 - 300 秒（默认 30 秒）
- 执行前确认（默认开启）

::: warning 注意
代码运行工具默认关闭，需手动开启。代码在本地执行，请注意安全。
:::

### 翻译

多语言翻译，支持以下翻译引擎：

| 引擎 | 需要 API Key |
|------|:---:|
| AI（使用当前模型） | 否 |
| DeepL | 是 |
| Google Translate | 是 |
| 百度翻译 | 是（AppID + Key） |

### 天气查询

查询城市天气信息。

| 数据源 | 需要 API Key |
|--------|:---:|
| wttr.in | 否 |
| OpenWeatherMap | 是 |

### 当前时间

返回当前日期、时间和时区。

## 使用方式

1. 在 **设置 > 工具** 中启用需要的工具
2. 在 **设置 > 工具** 中配置工具的最大调用轮数（默认 5 轮）
3. 对话时 AI 会根据需要自动调用工具

工具调用过程会在消息中实时显示状态。
