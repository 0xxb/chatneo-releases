# 快捷键

在 **设置 > 快捷键** 中查看和自定义。所有快捷键均可录制修改，支持冲突检测。

## 默认快捷键

### 对话管理

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 新建对话 | `⌘ N` | `Ctrl+N` |
| 删除对话 | `⌘ ⌫` | `Ctrl+Backspace` |
| 搜索对话 | `⌘ K` | `Ctrl+K` |
| 上一个对话 | `⌘ ↑` | `Ctrl+↑` |
| 下一个对话 | `⌘ ↓` | `Ctrl+↓` |

### 消息操作

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 聚焦输入框 | `⌘ L` | `Ctrl+L` |
| 重新生成 | `⌘ R` | `Ctrl+R` |
| 停止生成 | `Esc` | `Esc` |

### 界面

| 操作 | macOS | Windows / Linux |
|------|-------|-----------------|
| 切换侧边栏 | `⌘ S` | `Ctrl+S` |
| 打开设置 | `⌘ ,` | `Ctrl+,` |

## 发送键

在快捷键设置中可以切换消息发送键：

- **Enter** - 回车直接发送（默认）
- **⌘ Enter / Ctrl+Enter** - 回车换行，组合键发送
