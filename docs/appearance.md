# 外观设置

打开 **设置 > 外观** 自定义界面。

## 主题

- **跟随系统** - 自动匹配系统亮色 / 暗色
- **亮色模式**
- **暗色模式**

6 种主题色可选：默认、橙色、黄色、绿色、蓝色、粉色。

## 聊天背景

- 无背景（默认）
- 8 种预设渐变背景：暖日落、海洋蓝、森林绿、薰衣草、极光、夜空、玫瑰金、极简灰
- 自定义图片背景

背景可调节：
- 模糊度：0 - 20px
- 暗化度：0 - 100%

## 消息气泡

| 设置 | 说明 |
|------|------|
| 样式 | 扁平 / 气泡 |
| 不透明度 | 0 - 100% |
| 圆角 | 0 - 24px |

## 字体

- **正文字体** - 从系统字体中选择
- **代码字体** - 代码块使用的等宽字体
- **字号** - 小 / 中 / 大
- **行高** - 紧凑 / 标准 / 宽松

## 代码块

- **主题** - 自动（跟随明暗）、GitHub、One Dark、Monokai
- **自动换行** - 开启后长行代码自动换行

## 消息密度

控制消息之间的间距：紧凑 / 标准 / 宽松。

## macOS 特性

macOS 上支持侧边栏毛玻璃效果，与系统原生风格一致。
