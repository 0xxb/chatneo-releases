# 图片与视频生成

ChatNeo 支持通过 AI 模型生成图片和视频。

::: tip 专业版功能
图片和视频生成需要专业版。
:::

## 图片生成

选择支持图片输出的模型即可在对话中生成图片。

支持的提供商和模型：
- **OpenAI** - DALL-E 系列
- **Google** - Gemini 图片生成
- **xAI** - Grok 图片生成
- **Fireworks** - 图片生成模型

生成参数（可在模型设置中调整）：

| 参数 | 说明 |
|------|------|
| 尺寸 | 1024x1024 / 1536x1024 / 1024x1536 / 512x512 |
| 宽高比 | 1:1 / 16:9 / 9:16 / 4:3 / 3:4 |
| 生成数量 | 1 / 2 / 4 张 |

## 视频生成

目前支持通过火山引擎（字节跳动）生成视频。

配置火山引擎提供商后，选择视频生成模型即可在对话中生成视频。
