# 导出与截图

## 对话导出

将对话导出为多种格式。

::: tip
Markdown 导出免费可用，PDF 和 Word 导出需要专业版。
:::

### Markdown

导出内容包括：
- 对话标题和模型信息
- 角色标记（用户 / AI）
- 思考过程（可折叠）
- 消息正文
- 引用来源和相似度

### PDF

基于 HTML 渲染后转换为 PDF，保留完整样式。

### Word (DOCX)

导出为 Word 文档，包含：
- 格式化文本（标题、列表、表格）
- 嵌入图片
- 代码块（带语言标记）
- 引用块
- 超链接
- 思考过程
- 文件附件链接

## 对话截图

将对话渲染为 PNG 图片并自动复制到剪贴板。

三种截图模式：

| 模式 | 说明 |
|------|------|
| 单条消息 | 截图单条消息 |
| 单轮对话 | 截图一组用户-AI 消息 |
| 完整对话 | 截图整个对话 |

截图特性：
- 2x 高清渲染
- 继承当前聊天背景（渐变 / 图片 / 模糊）
- Mermaid 图表自动栅格化
- 自动写入剪贴板
