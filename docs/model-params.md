# 模型参数

在 **设置 > 高级 > 模型参数** 中配置全局默认参数。对话时也可以临时调整。

## 基础参数

| 参数 | 范围 | 说明 |
|------|------|------|
| Temperature | 0 - 2 | 控制回复随机性。0 最确定，2 最随机 |
| Max Output Tokens | - | 最大输出 Token 数 |

## 进阶参数

::: tip 专业版功能
以下参数需要专业版。
:::

| 参数 | 范围 | 说明 |
|------|------|------|
| Top-P | 0 - 1 | 核采样。与 Temperature 配合使用 |
| Top-K | 整数 | 每步只从概率最高的 K 个词中选择 |
| Frequency Penalty | -2 - 2 | 正值降低已出现词的重复率 |
| Presence Penalty | -2 - 2 | 正值鼓励模型讨论新话题 |
| Stop Sequences | 字符串列表 | 模型遇到这些序列时停止生成 |

## 专家参数

| 参数 | 说明 |
|------|------|
| Seed | 固定随机种子，使输出可复现 |
| Max Retries | 请求失败时的重试次数 |
| Timeout | 请求超时时间（毫秒） |
| Custom Headers | 自定义 HTTP 请求头（JSON 格式） |

## 参数优先级

1. 对话中临时调整的参数（最高优先级）
2. 全局默认参数
3. 模型默认值

## 模型能力目录

ChatNeo 内置模型能力目录（基于 LiteLLM），自动识别模型支持的功能（视觉、工具调用、推理等）。

可在 **设置 > 高级 > 模型参数 > 模型目录** 中手动刷新。

对于目录中未收录的模型，可以在模型设置中手动调整能力标记。
