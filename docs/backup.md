# 备份与同步

::: tip 专业版功能
备份恢复和 WebDAV 同步需要专业版。
:::

## 本地备份

打开 **设置 > 高级 > 数据**，可以选择性备份以下数据：

| 类别 | 说明 |
|------|------|
| 对话记录 | 所有对话和消息，包含附件文件 |
| 应用设置 | 全局设置项 |
| 提供商配置 | API Key、Base URL 等 |
| 自定义指令 | 系统提示词 |
| 插件 | 插件配置和启用状态 |
| MCP 服务 | MCP 服务配置 |
| 模型收藏 | 收藏的模型列表 |

备份生成 ZIP 文件，包含 JSON 数据文件和附件。

### 恢复

从 ZIP 文件恢复，恢复前可预览内容和选择恢复的类别。

::: warning 注意
备份文件可能包含 API Key 等敏感信息，请妥善保管。
:::

## WebDAV 云同步

支持通过 WebDAV 协议将数据同步到云端。兼容坚果云、NextCloud 等 WebDAV 网盘。

### 配置

在 **设置 > 高级 > 数据 > WebDAV 同步** 中填写：

| 字段 | 说明 |
|------|------|
| 服务地址 | WebDAV URL |
| 用户名 | 账号 |
| 密码 | 密码 |
| 备份路径 | 远程存储路径（默认 `/chatneo/backups/`） |

配置后可点击 **测试连接** 验证。

### 自动同步

| 设置 | 选项 |
|------|------|
| 同步间隔 | 1 小时 / 6 小时 / 12 小时 / 1 天 / 1 周 |
| 最大备份数 | 5 / 10 / 20 / 30 / 50 |

超出最大备份数时自动删除最早的备份。

### 操作

- 手动触发同步
- 查看云端备份列表
- 从云端恢复
- 删除云端备份
